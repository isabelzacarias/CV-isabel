Proyectos académicos destacados

GalaPerfecta - Sistema Cliente-Servidor (2025)

Desarrollo de una aplicación web para la gestión de eventos, con un enfoque en la interacción cliente-servidor a través de una API REST. La API fue creada en JavaScript, y la interfaz del cliente fue desarrollada en PHP con una base de datos MySQL.

  • Usuario: los usuarios podían consultar paquetes para eventos, obtener cotizaciones, realizar pagos de paquetes y consultar su historial de pagos.
  • Administrador: los administradores podían crear eventos, servicios y paquetes nuevos, así como gestionar usuarios mediante operaciones CRUD.

Además, se implementó JSON Web Token (JWT) para la validación de usuarios y la seguridad en las interacciones entre cliente y servidor.

Sistema de Gestion de Pizzeria - MVC (2024)

Desarrollo de una aplicación web para la gestión de una pizzería, con dos módulos principales: usuario y administrador.

  • Usuario: permitía visualizar los productos disponibles y consultar información detallada de cada pizza o producto.
  • Administrador: ofrecía la opción de agregar nuevos productos al catálogo, como pizzas u otros productos.

El sistema fue desarrollado en PHP utilizando el patrón MVC, y gestionaba los datos a través de una base de datos MySQL
